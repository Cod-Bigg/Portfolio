Planet Fact Site

A fact site for each planet

Overview

The purpose of this project was to put my react skills to use. It was a good challenge to practice using and changing state. Each of the planets Name and information were displayed by using them as props

Result

In the end result there is an 8 page fact site representing each planet. Both image and text chang when the user clicks on the overview, internal or surface buttons. The site was made using HTML5, CSS3, React, React-Router, and Bootstrap
